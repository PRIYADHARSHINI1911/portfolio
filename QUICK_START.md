## 🚀 Get Started in 3 Steps

### Step 1: View Locally
1. Open the `Portfolio` folder
2. Double-click `index.html`
3. Your portfolio opens in your browser!

### Step 2: Test It Out
- Click navigation links
- Scroll through sections
- Try the contact form
- Check on mobile (use browser dev tools: F12)

### Step 3: Deploy Online (Choose One)

#### **Easiest - GitHub Pages (5 minutes)**
1. Create GitHub account: github.com
2. Create new repo called `portfolio`
3. Upload your 3 files (html, css, js)
4. Go to Settings → Pages → Enable
5. Your site is live at: `github.io/portfolio`

#### **Fastest - Netlify (2 minutes)**
1. Go to netlify.com
2. Drag & drop your Portfolio folder
3. Done! Live at netlify.app

#### **Professional - Custom Domain**
1. Buy domain: namecheap.com ($0.88/year)
2. Point to GitHub Pages or Netlify
3. Your portfolio at: yourdomain.com

## 🎨 Customization Shortcuts

### Change Main Color:
In `styles.css`, find this and change the color:
```css
--primary-color: #6366f1;  /* Change to any color code */
```

### Add Your Photo (Optional):
Add an img folder with your photo, then add to HTML:
```html
<img src="img/photo.jpg" alt="Your Name" class="profile-pic">
```

### Modify Skills:
Find the Skills section and update these tags:
```html
<span class="skill-tag">Your Skill</span>
```
